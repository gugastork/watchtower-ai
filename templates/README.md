# Templates

Reusable project templates shared by the WatchTower community.

Structures and starting points for AI projects, PRDs and startup operations.

## Subdirectories

| Directory | What you will find |
|---|---|
| `content-templates/` | Templates for contributing to this repository |
| `prd/` | Product requirements document templates |
| `ai-projects/` | AI project starter templates |
| `startup-tools/` | Templates for startup operations |

## How to Contribute

Submit a PR with your template. See [CONTRIBUTING.md](../CONTRIBUTING.md) for full guidelines.
