# Content Templates

Use these templates when contributing to the WatchTower AI repository.

Each template is designed for a specific content type. Pick the one that matches your contribution, copy it, fill it in and submit your PR.

## Available Templates

| Template | Use for |
|---|---|
| [prompt-template.md](prompt-template.md) | Prompts for Claude Code, agents, research, marketing |
| [workflow-template.md](workflow-template.md) | AI workflows and automations |
| [case-study-template.md](case-study-template.md) | Real-world experiments and use cases |
| [tool-template.md](tool-template.md) | Plugins, CLI tools and open source utilities |
| [summary-template.md](summary-template.md) | Summaries of papers, tutorials and discussions |
| [resource-template.md](resource-template.md) | Curated resource collections |

## Tips

- Fill in every section — incomplete submissions may be asked to add more context
- Be honest about results and limitations
- Include real examples whenever possible
- Use clear, specific language
