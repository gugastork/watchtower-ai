# WatchTower AI

**The knowledge layer of the WatchTower builders community.**

WatchTower is a community of 2,000+ builders, developers and operators who build real things with AI — not talk about hype.

The problem: valuable knowledge shared in community discussions gets lost in chat flow. Prompts, workflows, experiments, tools — all buried in messages.

This repository exists to fix that. It captures, organizes and structures the best knowledge from the WatchTower community so builders can find, use and build on top of it.

---

## What Is Inside

| Directory | What you will find |
|---|---|
| [`prompts/`](prompts/) | Battle-tested prompts for Claude Code, agents, research and marketing |
| [`workflows/`](workflows/) | AI workflows for Claude Code, Cursor, n8n and automation tools |
| [`tools/`](tools/) | Plugins, CLI tools and open source AI utilities |
| [`templates/`](templates/) | Reusable templates for PRDs, AI projects and startup operations |
| [`case-studies/`](case-studies/) | Real experiments with results — what worked, what failed, why |
| [`summaries/`](summaries/) | Distilled insights from papers, tutorials and community discussions |
| [`resources/`](resources/) | Curated books, repos and articles worth your time |

## How to Use This Repo

**If you want to learn:**
Browse the directories above. Each folder has a README explaining what is inside. Start with the topic that matters most to your current project.

**If you want to contribute:**
Read [CONTRIBUTING.md](CONTRIBUTING.md). We have templates for each content type to make it easy. Every contribution goes through a review to keep quality high.

**If you want to discuss:**
Join the WatchTower community. Links below.

## From Prompting to Building

Most AI communities stop at prompts. WatchTower goes further.

The progression we care about:

```
prompting → workflows → agents → automation → real products
```

This repo reflects that progression. You will find content at every level — from a single useful prompt to full case studies of AI-powered products built by community members.

---

## Quality Standard

This is not a dump of random links and generic prompts.

Every contribution must:

- **Solve a real problem** — no theoretical exercises
- **Include context** — what it does, when to use it, what to expect
- **Be tested** — the author has actually used it
- **Be structured** — follow our content templates

See [CONTRIBUTING.md](CONTRIBUTING.md) for full guidelines.

## Origin

WatchTower started as a community of builders experimenting with Claude Code, AI agents and real-world AI workflows.

The initiative was created by **Gustavo Stork** — entrepreneur, product builder and founder of [The CMOs](https://thecmos.com.br) (one of the largest marketing leadership ecosystems in Brazil) and [Marketing.chat](https://marketing.chat) (an AI platform with 15,000+ users in 40+ countries).

The community grew organically through discussions about practical AI tools and quickly became one of the largest Claude-focused communities in Brazil.
