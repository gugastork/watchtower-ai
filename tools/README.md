# Tools

Plugins, CLI tools and AI utilities shared by the WatchTower community.

Tools that builders have created or recommend for real work.

## Subdirectories

| Directory | What you will find |
|---|---|
| `plugins/` | Editor and platform plugins |
| `cli/` | Command-line tools |
| `open-source/` | Open source AI tools and utilities |

## How to Contribute

Use the [tool template](../templates/content-templates/tool-template.md) and submit a PR. See [CONTRIBUTING.md](../CONTRIBUTING.md) for full guidelines.
