# Summaries

Distilled insights from papers, tutorials and community discussions.

The goal is to turn dense content into practical, actionable knowledge.

## Subdirectories

| Directory | What you will find |
|---|---|
| `papers/` | Summaries of AI research papers with practical takeaways |
| `tutorials/` | Tutorial summaries and guides |
| `community/` | Insights distilled from community discussions |

## How to Contribute

Use the [summary template](../templates/content-templates/summary-template.md) and submit a PR. See [CONTRIBUTING.md](../CONTRIBUTING.md) for full guidelines.
